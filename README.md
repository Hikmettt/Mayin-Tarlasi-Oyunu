*Genel Özelliği* <br/>

Klasik mayın tarlası oyunudur.Kendi isteğine göre boyutu değiştirip mayın sayılarını arttırabilirsiniz.<br/>
Zorluk seviyeleri 3 adettir. Kolay, orta ve uzman zorluk.<br/>
İlk başta butonlar yardımıyla zorluk seviyesi seçilir.
